EPIC Scotland Modelling

Welcome to the EPIC Scotland Modelling webpages. Here you will find the epidemiological modelling code developed by EPIC members.

Projects

PhylodynamicDataSets

PhylodynamicDataSets contains reference sequence data and trees for pathogens of interest to EPIC. It also contains some R utility script files used to generate the data where applicable. The data and html pages are in the gh-pages branch, so the PhylodynamicDataSets webpages are live at: http://epicscotland.github.io/PhylodynamicDataSets

Broadwick

Broadwick is a java framework for creating and running epidemiological models - this probject will be going live before 24th June 2014

BroadwickExamples

BroadwickExamples contains some simple stochastic compartmental models written within the Broadwick framework, get the code from https://github.com/EPICScotland/BroadwickExamples
